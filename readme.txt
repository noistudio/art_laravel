Laravel Better - библиотека для laravel с двумя разделенными модулями 
frontend и backend(админка)


После того как зальете все файлы в корневую директорию проекта
Ваш сервер должен поддерживать .htaccess MOD REWRITE,PHP7.3+,mysql,imagick
Перейдите по ссылке ВАШСАЙТ/install и будет визуальный установщик веб-сайта.

Библиотека умеет работать с mysql или mongodb 

Для  установки нужных таблиц  выполните миграцию. php artisan migrate

Выполните её в любом случае так как она и выполняет очищение ненужных данных.

И настройте данные . Пример .env файла в /engine/env_example 

Текущая версия сырая и работает сразу с mysql и mongodb 
Но отключить её можно, писать инструкцию лень,сами разберетесь.
