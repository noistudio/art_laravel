Laravel Better - библиотека для laravel с двумя разделенными модулями 
frontend и backend(админка)

Библиотека умеет работать с mysql или mongodb 

Для работы загрузите дамп с mysql из директории /engine/dump.sql

И настройте данные . Пример .env файла в /engine/env_example 

Текущая версия сырая и работает сразу с mysql и mongodb 
Но отключить её можно, писать инструкцию лень,сами разберетесь.
